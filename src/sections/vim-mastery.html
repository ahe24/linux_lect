실전 활용 예제 (Pro Tips)

Tip 1: 숫자 연속 자동 증가 (Sequence)

복사 붙여넣기 후, 숫자만 0, 1, 2...로 바꾸고 싶을 때 사용합니다.

Before
assign data[0] = 0;
assign data[0] = 0;
assign data[0] = 0;
assign data[0] = 0;
Action:
  1. 커서를 첫 번째 '0'에 위치
  2. Ctrl+v (Visual Block 모드 진입)
  3. jjj (아래로 3줄 확장하여 모든 '0' 선택)
  4. g Ctrl+a (선택한 숫자들을 0,1,2,3으로 증가)
* 감소는 g Ctrl+x
After
assign data[0] = 0;
assign data[1] = 1;
assign data[2] = 2;
assign data[3] = 3;
Tip 2: 모든 줄 끝에 세미콜론(;) 붙이기

줄 길이가 달라도 끝부분에 일괄적으로 문자를 추가할 수 있습니다.

Before
wire a 
wire [31:0] b
wire c_data
Action:
  1. 커서를 'wire'의 'w' 위치 (첫 줄)
  2. Ctrl+v (Visual Block 모드 진입)
  3. jj (아래로 2줄 확장, 세로 블록 선택)
  4. $ (블록을 각 줄 끝까지 확장)
  5. A (Append 모드) → ; 입력
  6. ESC (모든 줄에 적용)
After
wire a;
wire [31:0] b;
wire c_data;

5. 검색 및 일괄 치환 (Search & Replace)

변수명을 한 번에 바꿀 때 유용합니다. Verilog 리팩토링의 핵심입니다.

# 기본 검색 (Normal Mode)
/clk_en      # 아래로 'clk_en' 검색
?reset_n     # 위로 'reset_n' 검색
n / N        # 다음 / 이전 찾기

# 단어 바로 찾기
*            # 커서가 위치한 단어를 즉시 검색

# 치환 (Substitute)
# 문법: :[범위]s/찾을단어/바꿀단어/[플래그]

# 1. 파일 전체 치환 ('%'는 전체 파일을 의미)
:%s/wire/reg/g       # 파일의 첫 줄부터 끝 줄까지 모든 'wire' 변경

# 2. 현재 줄만 치환 ('%' 생략 시 커서가 있는 줄만 대상)
:s/wire/reg/g        # 현재 커서가 있는 줄의 'wire'만 변경

# 3. 특정 범위 지정
:10,20s/wire/reg/g   # 10번 줄부터 20번 줄까지만 변경

# Tip: 'g' (global) 플래그를 빼면 각 줄의 첫 번째 일치 항목만 바뀝니다.
# :%s/old/new        # (각 줄의 첫 번째 'old'만 'new'로 변경)

6. 매크로 (Macros): 반복 작업 자동화

포트 리스트의 끝부분 문자를 변경하거나 규칙적인 패턴을 수정할 때 강력합니다.

실전 예제: 줄 끝 콤마(,)를 세미콜론(;)으로 바꾸기

1. 실습용 텍스트 (VIM에 붙여넣으세요): 
wire [31:0] cpu_addr_0,
wire [31:0] cpu_addr_1,
wire [31:0] cpu_addr_2,
wire [31:0] cpu_addr_3,
wire [31:0] cpu_addr_4,
2. 매크로 녹화 및 실행 (순서대로 입력):
  1. 녹화 시작: qa (a 레지스터에 기록 시작)
  2. 작업 수행:
    • $ (줄 맨 끝으로 이동)
    • r ; (문자를 ;로 변경)
    • j (아래 줄로 이동)
    • 0 (줄 맨 앞으로 이동 - 중요!)
  3. 녹화 종료: q
  4. 반복 실행: 10@a (10번 반복)

7. .vimrc 설정 (엔지니어 필수)

홈 디렉토리의 ~/.vimrc 파일에 아래 내용을 추가하여 가독성을 높이세요.

" 문법 강조
syntax on

" 줄 번호 표시
set number

" 자동 들여쓰기 (Verilog 코딩 시 필수)
set autoindent
set cindent

" 탭 대신 스페이스 4칸 사용 (팀 코딩 컨벤션)
set expandtab
set tabstop=4
set shiftwidth=4

" 검색 시 대소문자 무시하되, 대문자 섞이면 구분
set ignorecase
set smartcase

" 괄호 짝 찾기 하이라이트
set showmatch
